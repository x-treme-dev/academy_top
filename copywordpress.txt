1. Создать новый сайт из архива с базой данных
2. На сайте-оргинале, который нужно скопировать, уставить плагин
для миграции All-in-One WP Migration
3. Через плагин экспортировать сайт в файл (с расширением .wpress)
4. На сайте-копии также нужно установить All-in-One WP Migration для импорта.
5. Нужно дать права на запись в директорию sudo chmod 777 wp-content
6. Создать файл .htaccess в корне сайта и записать туда директивы о размере загружаемых файлов
7. Сделать импорт сайта-оригинала на сайт-копию
8. Заходим на сайт-копию по пароле из сайта-оригинала
